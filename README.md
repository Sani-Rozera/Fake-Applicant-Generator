# Fake Applicant Generator

A Streamlit-based web application that generates realistic fake applicant profiles for recruitment practice, testing, and demo purposes. The app creates sample candidate data such as names, email addresses, phone numbers, skills, experience, resumes, and locations.

## Features

* Generate multiple fake applicant profiles instantly
* Realistic candidate details using random data generation
* View applicants in an interactive table
* Search and filter applicant records
* Download applicant data as CSV file
* Simple and user-friendly Streamlit interface

## Technologies Used

* Python
* Streamlit
* Pandas
* Faker Library

## Use Cases

* HR system testing
* Recruitment software demos
* Database practice projects
* Data analysis practice
* UI/UX prototype testing

## How to Run

1. Install required libraries:

pip install -r requirements.txt

2. Run the project:

streamlit run app.py

## Project Objective

The objective of this project is to automate the creation of dummy applicant data for testing and educational purposes while providing an easy-to-use dashboard for managing records.
